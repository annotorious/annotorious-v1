Annotation Test Page

This page serves as a development environment for the Yuma2 (working title) OKFN Annotator plugin. Hover over one of the images to get started. NOTE: make sure the plovr build tool is running! Use

java -jar plovr/plovr.jar serve okfn_plugin.json

to start plovr.
